My API Hub user interface

In its latest iteration my API Hub comes standard with a user interface. Most important technologies used for developing the UI are, besides JavaScript, React and Material UI. Because of the wonders of GitHub pages, the frontend is publicly available on the Internet. If you visit this page you can open JSON files containing collections of D&B Direct+ Data Blocks.

Currently (i.e. January '22) the same Direct+ Data Blocks are supported as in my previous, vanilla JavaScript implementation:

  • Company Information (@ level 1 & 2)
  • Principals & Contacts (@ level 1 & 2 & 3)
  • Hierarchies & Connections (@ level 1)
  • Financial Strength Insights (@ level 1 & 2)
  • Payment Insights (@ level 1)
  • Filings & Events (@ level 1)

The easiest way to obtain JSON files containing collections of Data Blocks is from here (alternatively you can download the entire repository as a zip archive). If you are in possession of Direct+ credentials it is possible to use Postman to download sets of Data Blocks for one or more DUNS of your own choice. My Postman collection of API requests is a good starting point in this case.

On GitHub you can see that the fictitious examples that I created are now almost a year old. In this time the layout of some D&B Data Blocks has changed. Over the next few weeks, I will, where necessary, update the examples and update the corresponding frontend code of version 4 of my API Hub. I have decided to no longer maintain my vanilla JavaScript project.
