Key concepts powering my blog

The three most important concepts behind my blog are;

  1. My blog is an ordered collection of blog posts,
  2. The blog posts are displayed in an HTML framework and
  3. JavaScript code makes end-user navigation possible.

This blog post is all about why my blog is an ordered collection of posts. Separate articles about the HTML(5) framework & JavaScript interactivity will follow.

In code the ordered collection of blog posts is implemented as a JSON array of article objects. The article object at index 0 references the blog index page. New blog posts can simply be added to the end of the array.

Event DOMContentLoaded is triggered when the blog is loaded.

In the DOMContentLoaded event handler a new navigation object is instantiated

In the navigation object's constructor the articles array defined in the JSON file is read.

A reference to the array is stored in property arrArticles

The navigation object also has a property for keeping track of which blog post is currently displayed, blogCurrIdx. This property can be used as an index value for array arrArticles to retrieve the article object associated with the currently active post.

Property blogCurrIdx keeps track of the currently displayed blog post

The file property of an article object should match up to an actual file in subdirectory src of the GitHub repository.

The blog post files are stored in directory src

The individual blog posts listed in subdirectory src are regular (albeit partial) HTML files.
