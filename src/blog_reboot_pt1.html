Blog reboot

It has been quiet on my blog for some time but, believe me, I've been working very hard. Looking back on the past couple of months it's easy to explain the absence of new posts. My previous post is all about deploying my Dockerized Node.js application on my laptop. At an earlier date I had already deployed the same code in a cloud container. These projects convinced me that I had successfully Dockerized my entire Node.js stack. Having come to this conclusion it was my intention to move on by (further) developing the application's user interface. As part of the journey I wanted to explore how far vanilla HTML-CSS-JavaScript would take me but, even after going the extra mile, I had to admit that this plan was just too time consuming. I'm therefore currently investing part of my time to learn React. I'm aware that this will probably turn out to be a long and winding road but please stay tuned for some (hopefully) interesting posts reIated to UI/React deveIopment.

While struggling with the vanilla HTML-CSS-JavaScript code I realized that I had to update the JavaScript notes I made a couple of years ago. Over time I've become a better JavaScript developer and now I'm very much aware that, even though the language might share certain constructs with languages I've used before (primarily C & C++), it is a completely different beast altogether. I therefore reread "JavaScript: The Definitive Guide (6th Edition)" by David Flanagan and made extensive notes on JSFiddle;

  1. Lexical Structure,
  2. Types, Values and Variables,
  3. Expressions and Operators,
  4. Statements
