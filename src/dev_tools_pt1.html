Setting up my dev environment

Software installs on Ubuntu invariably begin with: sudo apt update && sudo apt upgrade

I always install VIM first: sudo apt install vim

Compiling C or C++ code no longer is my daily business but I make it a point to install the GNU compilers: sudo apt install build-essential

There were times when I used SVN for version control but those days are over. Git now is the de facto standard: sudo apt install git

A quick check to verify if everything works as expected: cd ~/Documents/dev git clone https://github.com/hdr1001/cheat-sheets cd cheat-sheets/c gcc -c timestwo.c gcc -c basics.c gcc -o cbasics timestwo.o basics.o ./cbasics

Basically the same check but for C++: g++ -c basics.c timestwo.c g++ -o basics timestwo.o basics.o ./basics

In the process of verifying my code I made a couple of changes. To stage, commit & push these changes just: cd .. git add c/basics.c git commit -m "Changed placeholder for long unsigned int" git push origin
