Setting up my dev env, Node.js

As of the 26th of October 2021 version 16.13 is designated the active long term support (LTS) release of Node.js. Unfortunately Ubuntu's package management system is lagging a little bit behind. Even in Impish Indri (Ubuntu release 21.10) the supported Node package is still at version 12.22. That is just not good enough for my purposes. Luckily the binary distributions of Node are not hard to find. The version for Ubuntu is maintained by NodeSource and is located here. The installation instructions for version 16.x worked perfectly fine for me: curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -sudo apt-get install -y nodejsTo verify my install I executed:

With Node installed I immediately moved on to upgrading the Node Package Manager (npm) to the latest stable release. npm -vsudo npm install -g npm@latest

To test the install I build a Node.js and Express based prototype by:

  1. Creating a new code repository on GitHub,
  2. Cloning the repository and initializing the package manager,
  3. Adding the Express package as a dependency and
  4. Adding the JavaScript code.

Please note that all the commands necessary for creating the prototype are listed in commit comments on GitHub (just follow the links above ⬆️).
