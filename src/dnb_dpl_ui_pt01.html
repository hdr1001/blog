D&B Direct+ Data Blocks UI

D&B Direct+ has excellent documentation but sometimes having an example user interface implementation at your disposal can be a big help. In the past I have posted about persisting D&B Direct+ products and I have developed a React UI for Direct+ IDentity Resolution but I have never developed open source code to display the contents of D&B data products. That is exactly what I intend to do in January 2021. Of course, because of the introduction of Data Blocks functionality in the BeNeLux, this topic is especially newsworthy.

I do not want to dwell on my architectural and technical choices too long but I think it is important to mention that the project;

  1. will be developed using modern (vanilla) JavaScript, HTML5 & minimal CSS and
  2. is hosted on GitHub using GitHub Pages.

Today I created a new repository ...

... cloned the repository to my development PC ...

... added an index.html file to the project, inserted the Emmet HTML5 template & a reference to the Direct+ documentation site ...

... and, finally, enabled repository hosting using GitHub pages.

This way the project is literally live from day one!
