A React form hosted on GitHub

Over time I have created numerous, cheat sheet like forms. For example:

  • A very basic HTML4 form,
  • A similar, table based, basic XHTML form,
  • A CSS styled XHTML form,
  • React based forms hosted on JsFiddle and CodeSandbox.

When I was learning React I got in the habit of using code playgrounds like JsFiddle and CodeSandbox but the emergence of GitHub Pages changed all that. Having a local development environment which includes Visual Studio Code with the Live Server extension installed makes creating a lightweight, proof of concept type of project extremely easy. If, at some stage, either version control or online hosting of the project is needed, my go-to nowadays is GitHub.

That is how my cheat cheat form ended up hosted on GitHub pages: Because this is GitHub it is easy to follow along with the evolution of the code:

  1. Initial commit of a new repo on GitHub created using the standard GitHub UI,
  2. Code base after implementing the steps as described in the article titled "Create React App without create-react-app!" (details in the commit comments),
  3. I added the code necessary for deploying the form on GitHub pages. For more details just watch this Youtube video,
  4. In the remaining commits you can see how, more or less, one by one the controls are added to the form until, on the 8th of October 2021, I finalized the project.
