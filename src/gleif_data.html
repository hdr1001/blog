API Hub GLEIF data integration

GitHub has no issue with making it perfectly clear that you have been neglecting your project;

So I was looking for a short & sweet assignment to get at least one more green block on the board before the end of 2019. Integrating GLEIF data into my API Hub presented itself as a prime candidate. GLEIF has an extremely easy to use reference data API and my best guess was that I would able to get it implemented in a couple of hours.

Fortunately I was right and I even had some spare time left to do some other things as well on my day off. Commit 29 on GitHub provides an excellent overview of the code changes I had to make to integrate my first non-D&B API. In summary, a bit of configuration and SQL code and, voila, it is now possible to respond to GLEIF data requests as well. Just a quick reminder, it is standard Hub functionality to persist/archive the API's HTTP JSON reponse.
