Installing Ubuntu 18.04

My Lenovo Thinkpad T430 was handed to me at approximately the same time the new Long Term Support version of Ubuntu, Bionic Beaver (18.04), was released. Usually I don't install the OS before the 1st point release is made available but this time around I decided to live dangerously and opted to install the brand new version of the software.

I always download Ubuntu images from the mirror at the TU Delft but of course it's also possible to use the link on the operating system's official release page. Nowadays I use Etcher to create bootable USB drives.

To kick-off the actual install of Ubuntu Bionic Beaver I started my laptop with the USB drive plugged in and pressed the Enter key (↵) during the Power-On Self Test procedure. On a Lenovo Thinkpad this will take you to the Startup Interrupt Menu. Next, using function key F12, I accessed the Boot menu and booted my laptop from the USB drive. Compared to the LinuxTechi Ubuntu 18.04 Installation Guide there are a couple of other, less important, differences as well:

  1. My laptop has a UK keyboard (extended with Windows keys),
  2. I answered the question "Where are you?" with Amsterdam &
  3. I named my device RGDun.

Upon completion of the installation I rebooted my laptop. I unplugged the bootable USB drive when I was prompted to do so and, in order to force a restart from the ground up, pressed CTRL+ALT+DEL.

On reboot the PC's boot loader very neatly executed the Ubuntu Linux kernel (as deployed on my newly installed SSD) & took me to the desktop's GUI. After logging in I connected to my home WiFi network. I don't think I was able to connect during installation but at this stage things went very smoothly. With a working connection to the Internet I updated my OS: sudo apt-get update
sudo apt-get upgrade To finish things up I configured Canonical's Livepatch service. This service promises to apply critical kernel patches without rebooting. For Ubuntu community members this service is free for up to three PCs. Thank you Canonical!
