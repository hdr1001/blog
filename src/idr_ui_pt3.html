IDR UI, match candidate list

This week's assignment for me was to include a list of D&B Direct+ match candidates in my (X)HTML/CSS/JavaScript powered user interface. In the identity resolution process I view the specification & submission of the search criteria and the presentation of the match candidates as separate steps. My ambition was to reflect this in the user interface. On the Internet I stumbled upon this collapsible form element from the Materialize framework. It looks the part and probably works very well but for me the reason not to include it in my project was that it would create a dependency on the jQuery library. I've actually used this library quite a lot in the past but nowadays I'm very much convinced that you might not need it anymore.
