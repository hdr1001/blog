I still very much ❤️ Ubuntu

In 2021 Ubuntu still is my favorite Linux distribution (just as it was in 2017).It should therefore come as no surprise that I chose to install this Linux distro as the operating system for my new PC. Where, in the past, I tended to stick to long-term support releases I now decided to go for an interim release, Hirsute Hippo. At the time of this writing I have already moved on to the next interim release, Impish Indri. The plan is to move again when Jammy Jellyfish (22.04) becomes available and I expect to stay on that particular version of the software for a number of years as Jammy will be a LTS release.

My new PC will not have a dual boot option. Previously I always made sure that I could choose, on startup, whether to boot Linux or Windows. This time around though Ubuntu will be booted no matter what. I have reserved ample disk space though to deploy Windows if needed. When this scenario becomes relevant my plan is to deploy Windows as a virtual machine not as a separate boot option. In order to (potentially) facilitate this I have created my own, custom partition scheme;

Straight out of the box an Ubuntu install contains a slew of standard software components like FireFox, LibreOffice and the GNOME file manager. I usually immediately add Chromium for testing my web dev projects and GIMP for advanced image manipulation. In this case, because I have 24 CPU threads at my disposal, I installed Conky as well. On the right-hand side of this blog post you can see a screen recording of this system monitor in action on my PC. I have uploaded the configuration file to GitHub in case your interested. BTW the screen recording was made using Kazam and ffmpeg.
