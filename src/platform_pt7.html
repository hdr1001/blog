Constructing the platform, pt. VII

Before going any further it's probably a good idea to close some IP ports on my Droplet. I'm in no way a networking expert but luckily, on Digital Ocean's Ubuntu Droplets, UFW comes pre-installed. The goal of UFW is to make firewall management easy. Just start by checking if UFW is available out-of-the-box: ufw status verbose The status returned will probably be inactive but that's okay. In case UFW is not pre-installed my advise is to read Digital Ocean's excellent community write-up on this topic.

UFW should be configured to work with both IPv4 and IPv6. This can be checked in the main configuration file: vim /etc/default/ufw The relevant setting should look like this:

The firewall configuration can be accomplished in just a couple of simple steps: (1) disable UFW, (2) by default deny all incoming traffic and allow all outgoing traffic, (3) allow the following protocols: SSH, HTTP & HTTPS and (4) re-enable UFW.

Now reboot the Droplet to effectuate the new configuration: shutdown -r now
