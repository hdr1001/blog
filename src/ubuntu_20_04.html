Installing Ubuntu 20.04

If you follow my blog on a regular basis you know I'm big fan of the cloud services provided by Digital Ocean. However, recently I decided that, from now on, I will work on my personal projects on my Thinkpad T430 instead. Unfortunately, in March 2020, this laptop still ran on Ubuntu Bionic Bever (18.04). Usually I'm very conservative when it comes to installing a new OS. I tend to stick with long term releases and only install a new version of the Ubuntu LTS release a couple of months after the official launch. This time around though I opted to give Ubuntu Focal Fossa (20.04) a chance in pre-release and I'm happy to report that I wasn't disappointed!

The upgrade went without a hitch and I love the new look of my desktop (courtesy of GNOME v3.35). The Docker install done previously still worked like a charm. Of course I now host my API Hub repository on Github so I have updated my "deploy & run" gist.

After executing the steps listed above my project started immediately and without a single warning.
