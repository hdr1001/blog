When, as I do, you fill out forms on the Internet using your keyboard, you become aware of the importance of the order in which form elements receive focus. I'm always annoyed if this order is not logical. The same is true for a reset. In my modest opinion clicking the reset button should reset the form to its initial state and set the focus to the first input field.
